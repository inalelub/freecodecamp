Introduction

Java is a versatile, object-oriented programming language known for its portability and robustness. This documentation covers essential concepts in Java, including variable declarations, for loops, conditional statements, functions, arrays, and classes and objects.

Variable Declarations
1. Declaring Variables

In Java, you can declare variables to store data. To declare a variable, use the following syntax:

data_type variable_name;
  • `data_type`: The type of data the variable can hold (e.g., int, double, String).
  • `variable_name`: A unique indentifier for the variable.

Example:

int age; double price; String name;
2. Initializing Variables

You can also initialize variables when declaring them:

data_type variable_name = initial_value;

Example:

int count = 0; double temperature = 25.5; String greeting = "Hello, World!";
For Loops
1. The `for` Loop

A `for` loop is used for iterating over a range fo values or a collection of items. It consists of three parts:

  • Initialization: Setting an initial value.
  • Condition: Checking a condition to continue the loop.
  • Iteration: Modifying the value in each iteration.
for (initialzation; condition; iteration) { // Code to be executed in each iteration }

Example:

for (int i = 0; i < 5; i++) { System.out.println("Iteration " + i); }

This loop will execute the code block five times, printing "Iteration 0" to "Iteration 4."

2. Enhanced `for` Loop

The enhanced `for` loop simplifies iterating over arrays and collections. It does not require explicit initialzation and incrementation, making it ideal for iterating through collections.

for (data_type : array_or_collection) { // Code to be executed for each element }

Example:

int[] numbers = {1, 2, 3, 4, 5}; for (int num : numbers) { System.out.println(num); }
Conditional Statements
1. `if` Statement

The if statement allows you to execute code based on a condition. If the condition is true, the code within the block is executed.

if (condition) { // Code to be executed if the condition is true }
2. `else` Statement

The else statement can be combined with if to specify code that should be executed if the condition is false.

if (condition) { // Code to be executed if the condition is true } else { // Code to be executed if the condition is false }
3. `switch` Statement

The switch statement is used to evaluate a variable against multiple possible values and execute different code blocks based on the matched value.

switch (variable) { case value1: // Code to be executed if variable equals value1 break; case value 2: // Code to be executed if variable equals value1 break; default: // Code to be executed if variable equals value1 }
Functions
1. Defining Functions

Functions (also known as methods) are blocks of reusable code. You can define your own functions to perform specific tasks.

return_type function_name(parameters) { // Function code }
Arrays
1. Declaring Arrays

Arrays are collections of elements of the same data type. To declare an array, use the following syntax:

data_type[] array_name = new data_type[size];
2. Accessing Elements

You can access elements in an array using square brackets and the index of the element (starting from 0).

Classes And Objects
1. Class Definition

In Java, you can define classes to model real-world entities. A class defines attributes and methods that objects of the class can have.

class ClassName { // Class members (fields and methods) }
2. Creating Objects

To use a class, you need to create objects (instances) of that class.

ClassName objectName = new ClassName();
3. Object-Oriented Principles

Java follows object-oriented principles, including encapsulation, inheritance, and polymorphism, which enable modular and efficient programming. These principles are essential for designing complex software systems.
